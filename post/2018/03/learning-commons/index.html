京都大學的 Learning Commons (學習共享空間) 參訪感想

March 05, 2018
Travel

這次到京都自助旅行時因行程有經過京都大學，因此特別抽空參觀了校園以及圖書館，而在其中最讓我覺得有趣的是圖書館中的 Learning Commons（ラーニング・コモンズ，學習共享空間）的概念與功能。因此整理了一些 Learning Commons 的相關資料以及將參訪心得記錄下來。

何謂 Learning Commons？

Learning commons（學習共享空間）前身是被稱為 Information commos（資訊共享空間）的圖書館服務，最早概念為以數位資訊取代原先的紙本資訊，讓使用者能方便共享資訊。而現今由於資訊科技的進步，使得知識的傳遞與取得與更為便利，如 Notebook 或 Mobile Phone 等已經能產生過去 Information Commons 所提供的資訊共享功能。因此 Information Commons 的概念被進一步提升，將知識的傳達轉換到知識的創造與自主的學習，也就是對學習支援功能的特化，而誕生了 Learning Commons 的概念與功能。

京都大學的 Learning Commons

這次所參觀的京都大學的圖書館雖然只能暫時以見學的身份進入，參觀區域也有限也因為禁止攝影只能敘述所見的部分（但仍然能進入 Learning Commons 區域），以下是個人看到與其他大學圖書館區域不同的地方

  • 開放的共同空間，與一般圖書館內所見的閱覽室或是討論用小房間不同，內部可看到許多學生自習或使用白板討論數學問題等。
  • 不論是椅子，桌子，白板等都是可以自由移動的，可視需要自由移動與調整討論位置。
  • 區域隔音好，就算大聲討論也不會影響到外部的圖書館安寧。
  • 有公告說可以拿一個牌子放在桌上鼓勵其他人加入討論，這種在其他場所沒見過的作法。
  • 可申請學習支援（学習サポートデスク），可先提出問題後與特定領域的研究生進行討論，以協助學生解決學習問題。

京都大學附屬圖書館的 Learning Commons 區域 (From: Here )

在看過這些設備和學習方式後，個人認為這是一種新型態且能有更多收穫的學習方式。相較於過去較多是閉鎖的知識傳達（如圖書閱讀或小組內的討論），這種開放空間更能將資訊散播到不同的接收者身上，不論是否是該領域的學生都能夠廣泛接收到其他資訊，進而幫助知識的產出。

而除了京都大學外，日本大學近幾年也導入了 Learning Commons 區域，如了下方的東京都市大學與大阪大學外，上智大學四谷校區的圖書館就在 2009 年時開始營運的 Learning Commons 區域是個相當著名的範例。而台灣的大學圖書館也有建立 Learning Commons 區域，但就個人所知大部分似乎只到達 Information Commons，也就是雖然有設置了相關如電腦網路等設備，但並沒有到達學習支援的階段，算是相當可惜的部分。
